# v0.3.1 3端末動作確認チェック表

対象:

- Windows 11 / Microsoft Edge
- iPhone / Safari（通常タブとホーム画面PWAを区別）
- Android / Google Chrome（通常タブまたはインストール済みPWA）

## 事前確認

- [ ] GitHub Pagesのデプロイが成功している
- [ ] 記録タブのアプリ版が `v0.3.1`
- [ ] 変更前の学習データをJSONで書き出した
- [ ] iPhoneは普段利用する形態（Safariまたはホーム画面PWA）で開いた

## 共通機能

- [ ] ホーム、学習、カード、ラボ、記録の全タブが表示される
- [ ] 既存の回答数・正答率・復習予定が引き継がれている
- [ ] 新規問題を1問回答し、採点・解説・次回復習日が表示される
- [ ] 用語カードのフリガナ、日本語訳が表示される
- [ ] 数式カードの読み方が表示される
- [ ] 比較カードが表示される

## カメラ・動画ラボ

- [ ] カメラ許可後に元映像と処理後映像が表示される
- [ ] 一時停止後、元映像をタップすると畳み込みインスペクタが出る
- [ ] 動画ファイルを選択できる
- [ ] 再生・停止・コマ送りが動く
- [ ] フレーム差分で動いた部分が強調される
- [ ] 背景差分、時間平均、エッジ、解像度、fpsの各モードを選べる
- [ ] タブを離れるとカメラが停止する

## データ管理

- [ ] JSONバックアップをダウンロードできる
- [ ] バックアップの読み込み前に件数確認が出る
- [ ] 問題JSONの不正データが「検査」で拒否される
- [ ] 正しい問題JSONで新規・更新件数が表示される
- [ ] 問題取り込み前にバックアップがダウンロードされる
- [ ] 端末・PWA診断を実行できる

## PWA・オフライン

- [ ] オンラインで一度開いた後、機内モードでもホーム画面が開く
- [ ] オンラインへ戻すと最新版へ更新される
- [ ] iPhoneのSafariとホーム画面PWAで保存領域が別であることを認識している

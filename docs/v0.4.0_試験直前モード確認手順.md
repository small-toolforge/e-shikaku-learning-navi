# v0.4.0 試験直前モード確認手順

対象版: `v0.4.0-dev.20`

## 目的

15分集中モードが、未回答問題を不正解として数えず、回答済み問題だけで結果を集計することを確認します。

## 基本動作

1. ホームまたは学習画面で「15分集中（最大15問）」を押す
2. 問題画面上部に`15:00`から始まるタイマーが表示される
3. 問題に「出題対象」または「出題範囲混在」のタグが表示される
4. 2問回答する
5. 上部の「ここまでの結果」を押す
6. 確認画面で続行する

期待結果:

```text
回答: 2問
未回答: 13問
正解数: 0～2 / 2
回答内正答率: 回答した2問だけで計算
終了理由: 途中終了
```

未回答13問は不正解として扱いません。

## 同じ問題でもう一度

結果画面で「同じ問題でもう一度」を押します。

- 同じ15問で再開する
- タイマーが15:00へ戻る
- 前回の回答結果は学習履歴として保持される
- 新しいセッションの回答数は0から始まる

## 全問完了

15問すべて回答すると、終了理由は「全問完了」になります。

期待結果:

```text
回答: 15問
未回答: 0問
正解数: 正解数 / 15
```

## 制限時間終了

15分経過時の仕様:

- 問題回答前に時間切れになっても、その問題は回答できる
- 「ここまでの結果」は「この問題を回答後に終了」へ変わり、押せなくなる
- 回答と採点後、次ボタンが「時間終了の結果を見る」へ変わる
- 結果画面の終了理由は「制限時間終了」
- 未回答問題は不正解として数えない

時間切れ確認は最終受け入れ時に1回実施します。通常の回帰確認では、途中結果と全問完了の確認を優先します。

## 保存への影響

回答済み問題だけが次へ保存されます。

- 回答履歴
- 自信度
- 誤答原因
- 問題の復習予定

未回答問題には回答ログも復習予定も作成しません。

## 確認結果

- [ ] タイマーが15:00から減る
- [ ] 最大15問で開始する
- [ ] オプションだけの問題を除外する
- [ ] 途中結果ボタンが動く
- [ ] 回答数を分母にして正答率を計算する
- [ ] 未回答を不正解に数えない
- [ ] 回答・未回答・回答内正答率・経過時間が表示される
- [ ] 同じ問題でもう一度が動く
- [ ] 全問完了を判定できる
- [ ] 制限時間終了後、回答中の問題を破棄しない

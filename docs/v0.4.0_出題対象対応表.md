# v0.4.0 出題対象・オプション対応表

対象: JDLA E資格 試験出題範囲（シラバス2026）第1.0版

## 判定方針

公式PDFでは「グレー網掛け：オプション（出題対象外）」と定義されています。本アプリでは次の3状態で表示します。

- 出題対象: 対応する小項目が白地であるもの
- オプション（出題対象外）: 対応する小項目全体がグレーであるもの
- 出題対象・オプション混在: 同一小項目内で白地とグレー網掛けが混在するもの

混在項目は、カード単位の誤分類を避けるため、安全側に「混在」と表示します。混在項目を出題対象だけに絞り込む細粒度対応は次版候補です。

## UI仕様

カードごとに次のタグを表示します。

- 出題対象
- オプション（出題対象外）
- 出題対象・オプション混在

カード画面には出題範囲フィルターを設けます。初期値は「すべて表示」とし、オプションを自動的に非表示にはしません。これは混在項目の誤除外を防ぐためです。

## 今後の精密化

混在項目について、公式PDFの細項目・キーワード単位でカードIDへ直接`scope`を付与します。精密化後は次の動作を検討します。

- 試験直前モードでは出題対象のみを既定表示
- オプションカードは補足学習として折りたたむ
- 問題にもscopeを付け、出題対象問題だけの模擬セッションを作る

## 注意

この対応表は公式PDFの視覚的な網掛けを基準に作成しています。PDFのテキスト抽出結果だけでは背景色が失われるため、改訂版PDFが出た場合は画像表示でも再確認します。

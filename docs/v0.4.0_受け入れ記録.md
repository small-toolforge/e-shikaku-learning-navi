## v0.4.0-dev.18 カード理解度確認

Windows 11・Edge・ローカルHTTPで、数学的基礎の用語カードを使って確認しました。

確認操作:

- `Matrix / 行列`を「苦手」に設定
- `Tensor / テンソル`を「曖昧」に設定
- 「本日までの復習対象だけ」をON
- 再読み込み後、理解度と復習予定が保持されることを確認
- `Matrix / 行列`を「覚えた」へ変更し、3日後から7日後へ間隔が延長されることを確認
- 記録画面のカード理解度集計を確認

確認結果:

- [x] 苦手を選ぶとカードに「今日復習」と表示
- [x] 曖昧を選ぶとカードに「明日復習」と表示
- [x] 集計が`苦手1・曖昧1・覚えた0・本日復習1・未設定267`へ更新
- [x] 本日までの復習対象だけにすると`Matrix / 行列`の1枚だけを表示
- [x] 選択状態が赤系・黄系で識別できる
- [x] 再読み込み後も理解度と復習予定が保持される
- [x] 「覚えた」を続けて選ぶと復習間隔が3日から7日へ延長される
- [x] 記録画面の集計が`苦手0・曖昧1・覚えた1・本日復習0`と一致する

集計の意味:

- カード下部の`復習N回`は、そのカードで理解度ボタンを記録した累計回数
- カード画面上部と記録画面の`苦手・曖昧・覚えた`は、現在その状態にあるカード枚数
- 状態を`苦手`から`覚えた`へ変更すると、苦手の枚数は減り、覚えたの枚数が増える

初回セルフチェックで、次の3問に存在しないカード参照が見つかりました。

| 問題ID | 修正前 | 修正後 |

# v0.4.0 論文図解アトラス 基盤実装メモ

## 今回の実装範囲

- カード画面へ「論文図解」タブを追加
- Transformerを第一号として自作SVGで図解
- 全体・Encoder・Decoderの3表示
- 各ブロックのタップ／キーボード操作による解説切替
- 英語の発音用フリガナ、日本語訳、試験での見分け方
- 原典論文リンク
- Transformer確認問題3問を既存の間隔反復へ接続
- シラバス2026の索引、キーワード検索、大分類フィルター
- 後続図解の紐付け先を索引へ登録

## 版の扱い

現在は開発ブランチ上の `v0.4.0-dev.1` です。安定版の `main` と `v0.3.1` は変更しません。

アプリ全体の診断画面は、正式な版更新まで `v0.3.1` と表示されます。論文図解画面には `v0.4.0-dev.1` を表示します。

## シラバス索引

JDLA E資格シラバス2026の小項目・細項目・キーワードを検索可能なデータとして登録しています。

今回の段階では、グレー網掛けで示されるオプション（出題対象外）の個別フラグは未確定です。原資料の網掛けを再確認したうえで次の工程で付与します。

## 検査

Windowsでは次を実行します。

```text
tools\v0.4.0_基盤検査.cmd
```

Anaconda Promptでは次を直接実行できます。

```text
python tools/v0.4.0_基盤検査.py
```

主な確認内容:

- 新旧CSS・JavaScriptの参照順
- Transformer図解ノード
- 確認問題3問
- シラバス索引件数
- Service Workerのキャッシュ対象
- Node.jsがある場合のJavaScript構文

## 実機確認予定

1. Windows 11 / Microsoft Edge
2. iPhone / ホーム画面PWA
3. Android / ホーム画面PWA

確認項目:

- 既存のv0.3.1機能が壊れていないこと
- 「カード → 論文図解」が開くこと
- 全体／Encoder／Decoder表示が切り替わること
- 各ブロックのタップで解説が切り替わること
- 確認問題3問が回答履歴と復習予定へ入ること
- シラバス索引で `RAG`、`量子化`、`マハラノビス距離` を検索できること
- オフライン再起動後も図解が開くこと

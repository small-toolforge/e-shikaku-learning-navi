# v0.4.0 論文図解・シラバスカード 実装メモ

## 現在の開発版

`v0.4.0-dev.5`

安定版の `main` と公開中の `v0.3.1` は変更していません。開発は `work/v0.4.0-atlas-foundation` とDraft PR #9で進めます。

## 論文図解

Transformerを含む12図解を収録しています。

1. Transformer
2. ResNet / WideResNet
3. Vision Transformer / Swin Transformer
4. Faster / Mask R-CNN・YOLO / SSD・FCOS
5. FCN / U-Net
6. Word Embedding
7. BERT / GPT / RAG
8. STFT / Mel / MFCC・WaveNet・CTC
9. Autoencoder / VAE / GAN / Diffusion / Flow
10. DQN / A3C / Actor-Critic
11. Transfer / Self-Supervised / Active / Metric / Meta Learning
12. Grad-CAM / Integrated Gradients / LIME / SHAP

各図解には、自作SVG、発音用フリガナ、日本語訳、処理順、試験での見分け方、一言対比、原典・代表論文、確認問題への導線を付けています。

### 2. 機械学習

`v0.4.0-dev.5`で次の125枚を追加しました。

- 用語カード：76枚
- 数式カード：29枚
- 比較カード：20枚

対象範囲:

- 最近傍法、k近傍法、kd-tree、近似最近傍
- ユークリッド・マンハッタン・Lp・コサイン・マハラノビス距離
- 教師あり・教師なし・半教師あり学習
- 線形回帰、最小二乗法、Lasso、Ridge、L1・L2正則化
- ロジスティック回帰、ロジット、シグモイド、ソフトマックス、オッズ
- SVM、サポートベクター、ハード／ソフトマージン、カーネル法
- 決定木、CART、Gini不純度、Bagging、Random Forest、Boosting
- PCA、寄与率、SNE、t-SNE、Crowding Problem
- k-means、階層的クラスタリング、デンドログラム、ウォード法、群平均法
- 過剰適合・過少適合、Bias / Variance、次元の呪い
- Holdout、k-Fold Cross-Validation
- Accuracy、Precision、Recall、F1、ROC、AUC、IoU、mAP
- micro / macro平均、MSE、RMSE、MAE、Perplexity

数式カードには、日本語での読み方、意味、覚え方、記号、具体例、よくある誤りを付けています。

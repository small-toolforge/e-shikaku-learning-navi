## 論文図解

Transformerを含む12図解を収録しています。

1. Transformer
2. ResNet / WideResNet
3. Vision Transformer / Swin Transformer
4. Faster / Mask R-CNN・YOLO / SSD・FCOS
5. FCN / U-Net
6. Word Embedding
7. BERT / GPT / RAG
8. STFT / Mel / MFCC・WaveNet・CTC
9. Autoencoder / VAE / GAN / Diffusion / Flow
10. DQN / A3C / Actor-Critic
11. Transfer / Self-Supervised / Active / Metric / Meta Learning
12. Grad-CAM / Integrated Gradients / LIME / SHAP

各図解には、自作SVG、発音用フリガナ、日本語訳、処理順、試験での見分け方、一言対比、原典・代表論文、確認問題への導線を付けています。

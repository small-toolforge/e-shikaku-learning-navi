## シラバスとの対応

JDLA E資格シラバス2026の小項目・細項目・キーワードを検索可能な索引として登録しています。

カードデータは次の安定ID形式です。

```text
term-4-4-rag
formula-4-7-qlearning
compare-4-4-bert-gpt
```

現在は全カードを「シラバス掲載」と表示します。グレー網掛けで示されるオプション（出題対象外）は、原本との対応表を確定してから個別フラグを付与します。誤った出題対象表示を避けるため、この段階では断定しません。

## ローカル確認の起動

次のCMDを使用します。

```text
tools\v0.4.0_ローカル確認.cmd
```

CMDはWindowsの文字コード差による誤動作を避けるため、本文をASCII英数字だけで記述しています。GitHub Pagesの公開動作には関係しないローカル補助ファイルです。

Anaconda Promptから直接起動する場合:

```text
python -m http.server 8000 --bind 127.0.0.1 --directory ..
```

接続先:

```text
http://127.0.0.1:8000/
```

参照修復ファイルは、問題データの読込み後かつカードへの相互リンク生成前に実行します。

```text

  ↓

```

既にSeed版7を利用している端末にも反映するため、Seed版8で上記3問だけを上書きします。回答履歴、問題SRS、自信度、誤答原因は別ストアに保存されているため維持します。

- 修復先カードIDが実在すること
- 修復処理が問題統合より前に読み込まれること

## カード理解度と間隔反復

各シラバスカードへ次の3段階を追加しました。

| 状態 | 次回復習 |
|---|---|
| 苦手 | 当日 |
| 曖昧 | 翌日 |
| 覚えた | 3日 → 7日 → 14日 → 30日 → 60日 → 90日 |

「覚えた」を続けて選ぶと復習間隔を段階的に延ばします。「曖昧」へ変更すると翌日、「苦手」へ変更すると当日の復習対象へ戻します。

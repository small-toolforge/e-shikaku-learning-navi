# v0.4.0 論文図解・シラバスカード 実装メモ

## 現在の開発版

`v0.4.0-dev.4`

安定版の `main` と公開中の `v0.3.1` は変更していません。開発は `work/v0.4.0-atlas-foundation` とDraft PR #9で進めます。

## 論文図解

Transformerを含む12図解を収録しています。

1. Transformer
2. ResNet / WideResNet
3. Vision Transformer / Swin Transformer
4. Faster / Mask R-CNN・YOLO / SSD・FCOS
5. FCN / U-Net
6. Word Embedding
7. BERT / GPT / RAG
8. STFT / Mel / MFCC・WaveNet・CTC
9. Autoencoder / VAE / GAN / Diffusion / Flow
10. DQN / A3C / Actor-Critic
11. Transfer / Self-Supervised / Active / Metric / Meta Learning
12. Grad-CAM / Integrated Gradients / LIME / SHAP

各図解には、自作SVG、発音用フリガナ、日本語訳、処理順、試験での見分け方、一言対比、原典・代表論文、確認問題への導線を付けています。

### 1. 数学的基礎

`v0.4.0-dev.4`で次の70枚を追加しました。

- 用語カード：41枚

対象範囲:

- 行列、テンソル、アダマール積、ランク、勾配
- 固有値・固有ベクトル・対角化
- 特異値分解
- 同時確率・条件付き確率・周辺確率
- PMF / PDF、期待値、分散、共分散
- ベルヌーイ・二項・カテゴリカル・ガウス・t分布・混合ガウス
- 中心極限定理
- ベイズ則、ナイーブベイズ、対数尤度、MLE、MAP、ベイズ推定
- 自己情報量、相互情報量、各種エントロピー
- KL / JSダイバージェンス

数式カードには、日本語での読み方、意味、覚え方、記号、具体例、よくある誤りを付けています。


```text

```

## ローカル確認の起動

`tools` フォルダ内で単純に `python -m http.server 8000` を実行すると、`tools` フォルダ自体が配信されるため、アプリではなく検査ファイルの一覧が表示されます。

次のCMDを使用してください。

```text
tools\v0.4.0_ローカル確認.cmd
```

このCMDは、実行場所に関係なくリポジトリのルートを `127.0.0.1` のみに公開します。Edgeで次を開きます。

```text
http://127.0.0.1:8000/
```

Anaconda Promptから直接起動する場合は、`tools` フォルダ内でも次のコマンドなら正しく動作します。

```text
python -m http.server 8000 --bind 127.0.0.1 --directory ..
```

終了するときは、起動した画面で `Ctrl+C` を押します。

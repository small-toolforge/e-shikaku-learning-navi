# E資格 学習ナビ v0.3.1 リリースノート

リリース判定日: 2026-07-11

## 概要

v0.3.1は、既存のv0.3.0を基点として、データ保存、問題JSON取込、バックアップ、PWA診断、3端末互換性を中心に安定化した版です。

Windows 11、iPhone、Androidの実機確認を完了し、安定版として受け入れました。

## 主な改善

- 正規実装を版固定の`assets/v0.3.1/`へ整理
- 旧試作ファイルを公開経路から除外
- 問題JSONの全件事前検査、件数プレビュー、一括保存
- 回答ログと復習予定の原子的保存
- 採点ボタン連打による二重登録の防止
- 問題取込、バックアップ読込、履歴削除前の復元用スナップショット
- バックアップ統合時の復習予定の新旧判定
- 最終バックアップ日の案内
- IndexedDB、Service Worker、カメラ、Canvas、保存容量の端末診断
- Windows 11、iPhone、Android向けの動作確認手順
- Anaconda環境を含むリリース検査CMDの実行環境探索

## 互換性

- IndexedDB名`eshikaku_v1`を維持
- 既存問題ID、回答ログ、復習予定を引き継ぐ
- 外部ライブラリとビルド工程は追加しない
- カメラ映像とローカル動画は端末内で処理し、送信しない

## 実機確認

| 端末 | 利用形態 | 判定 |
|---|---|---|
| Windows 11 | Microsoft Edge | 合格 |
| iPhone | ホーム画面PWA | 合格 |
| Android | ホーム画面PWA | 合格 |

確認内容:

- 基本学習機能
- 用語、数式、比較カード
- カメラ・動画ラボ
- IndexedDB保存
- Service Workerとオフライン利用
- JSONバックアップの書き出し・読み込み
- iPhoneとWindows 11間のデータ移行

詳細は`docs/3端末動作確認チェック表.md`を参照してください。

## 既知の注意事項

- iPhoneのSafari通常タブとホーム画面PWAは、保存領域が別になる場合があります。通常利用はホーム画面PWAへ統一してください。
- ブラウザデータの削除や端末故障に備え、JSONバックアップを定期的に保存してください。
- Node.jsは動作要件ではありません。未導入の場合、リリース検査ではJavaScript構文検査だけが警告扱いで省略されます。
